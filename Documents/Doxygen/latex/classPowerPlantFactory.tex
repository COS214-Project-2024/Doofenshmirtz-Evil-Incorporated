\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classPower}{Power}} Plant buildings in the city simulation. 

A factory class responsible for creating \mbox{\hyperlink{classPower}{Power}} Plant buildings.

The \mbox{\hyperlink{classPowerPlantFactory}{Power\+Plant\+Factory}} class implements the factory method to create instances of the \mbox{\hyperlink{classPower}{Power}} Plant building type and provides resource cost details for building power plants.

The \mbox{\hyperlink{classPowerPlantFactory}{Power\+Plant\+Factory}} class derives from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides the implementation details for creating a \mbox{\hyperlink{classPower}{Power}} Plant building, retrieving its construction cost, and specifying the resources required.
