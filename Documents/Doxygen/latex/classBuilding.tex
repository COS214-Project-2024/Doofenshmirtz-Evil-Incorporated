\doxysubsection{Detailed Description}
Represents a building within a city unit. 

The \mbox{\hyperlink{classBuilding}{Building}} class inherits from \mbox{\hyperlink{classCityUnit}{City\+Unit}} and provides functionality for managing residents, calculating capacities, and evaluating various building metrics.
