\doxysubsection{Detailed Description}
Represents a waste management utility building. 

This class inherits from the \mbox{\hyperlink{classUtility}{Utility}} class and implements the functionality specific to waste management, including methods to update the waste site\textquotesingle{}s status and collect relevant resources.
