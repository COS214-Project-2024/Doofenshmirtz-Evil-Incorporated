\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classWaste}{Waste}} Site buildings in the city simulation. 

A factory class responsible for creating \mbox{\hyperlink{classWaste}{Waste}} Site buildings.

The \mbox{\hyperlink{classWasteSiteFactory}{Waste\+Site\+Factory}} class implements the factory method to create instances of the \mbox{\hyperlink{classWaste}{Waste}} Site building type and provides resource cost details for building waste management sites.

The \mbox{\hyperlink{classWasteSiteFactory}{Waste\+Site\+Factory}} class derives from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides the implementation details for creating a \mbox{\hyperlink{classWaste}{Waste}} Site building, retrieving its construction cost, and specifying the resources required.
