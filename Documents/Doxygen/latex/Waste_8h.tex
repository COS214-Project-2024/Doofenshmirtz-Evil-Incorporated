\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classWaste}{Waste}} class, representing a waste management utility building. 

The \mbox{\hyperlink{classWaste}{Waste}} class extends the \mbox{\hyperlink{classUtility}{Utility}} class and is responsible for managing waste resources within the city. It provides methods to update the state of the waste site, collect waste resources, and retrieve utility-\/related data in JSON format.
