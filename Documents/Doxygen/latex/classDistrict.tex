\doxysubsection{Detailed Description}
A class that represents a district composed of multiple City\+Units. 

Inherits from \mbox{\hyperlink{classCityUnit}{City\+Unit}} and manages a collection of nested city units. The class provides functionality for adding, removing, and managing City\+Units, as well as policy-\/based updates and resource collection.
