\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classIndustrial}{Industrial}} buildings in the city simulation. 

A factory class responsible for creating \mbox{\hyperlink{classIndustrial}{Industrial}} buildings.

The \mbox{\hyperlink{classIndustrialFactory}{Industrial\+Factory}} class implements the factory method to create instances of the \mbox{\hyperlink{classIndustrial}{Industrial}} building type and provides resource cost details for building industrial structures.

The \mbox{\hyperlink{classIndustrialFactory}{Industrial\+Factory}} class derives from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides the implementation details for creating an \mbox{\hyperlink{classIndustrial}{Industrial}} building, retrieving its construction cost, and specifying the resources required.
