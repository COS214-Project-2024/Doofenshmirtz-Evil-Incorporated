\doxysubsection{Detailed Description}
Base class for commands that can be executed on a \mbox{\hyperlink{classCityUnit}{City\+Unit}}. 

This abstract class defines an interface for commands that can be used to implement various government policies within a city unit (e.\+g., a district).
