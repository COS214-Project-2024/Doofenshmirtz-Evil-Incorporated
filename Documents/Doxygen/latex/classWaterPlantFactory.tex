\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classWater}{Water}} Plant buildings in the city simulation. 

A factory class responsible for creating \mbox{\hyperlink{classWater}{Water}} Plant buildings.

The \mbox{\hyperlink{classWaterPlantFactory}{Water\+Plant\+Factory}} class implements the factory method to create instances of the \mbox{\hyperlink{classWater}{Water}} Plant building type and provides resource cost details for building water treatment facilities.

The \mbox{\hyperlink{classWaterPlantFactory}{Water\+Plant\+Factory}} class derives from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides the implementation details for creating a \mbox{\hyperlink{classWater}{Water}} Plant building, retrieving its construction cost, and specifying the resources required.
