\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classResidential}{Residential}} buildings in the city simulation. 

A factory class responsible for creating \mbox{\hyperlink{classResidential}{Residential}} buildings.

The \mbox{\hyperlink{classResidentialFactory}{Residential\+Factory}} class implements the factory method to create instances of the \mbox{\hyperlink{classResidential}{Residential}} building type and provides resource cost details for building residential structures.

The \mbox{\hyperlink{classResidentialFactory}{Residential\+Factory}} class derives from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides the implementation details for creating a \mbox{\hyperlink{classResidential}{Residential}} building, retrieving its construction cost, and specifying the resources required.
