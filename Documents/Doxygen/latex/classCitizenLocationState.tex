\doxysubsection{Detailed Description}
Abstract class representing a state for a \mbox{\hyperlink{classCitizen}{Citizen}}\textquotesingle{}s location. 

This class defines the interface for different location states (e.\+g., \mbox{\hyperlink{classAtHomeState}{At\+Home\+State}}, \mbox{\hyperlink{classAtWorkState}{At\+Work\+State}}, etc.) of a citizen in the simulation. The travel function is virtual and must be implemented by derived classes.
