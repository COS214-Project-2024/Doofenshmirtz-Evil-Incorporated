\doxysubsection{Detailed Description}
Abstract factory class for creating buildings. 

The \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} class defines the interface for creating building objects. Derived classes are responsible for defining specific types of buildings and their associated costs.
