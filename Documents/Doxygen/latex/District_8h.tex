\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classDistrict}{District}} class for managing a collection of \mbox{\hyperlink{classCityUnit}{City\+Unit}} objects. 

The \mbox{\hyperlink{classDistrict}{District}} class is a composite structure that allows for the management of multiple \mbox{\hyperlink{classCityUnit}{City\+Unit}} objects, such as buildings and landmarks, within a city simulation. It provides methods for adding, removing, and updating units, as well as collecting resources and utilities.
