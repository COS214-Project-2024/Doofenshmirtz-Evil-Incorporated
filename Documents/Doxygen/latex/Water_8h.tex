\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classWater}{Water}} class, representing a water utility building. 

The \mbox{\hyperlink{classWater}{Water}} class extends the \mbox{\hyperlink{classUtility}{Utility}} class and is responsible for managing water resources within the city. It provides methods to update the state of the water plant, collect water resources, and retrieve utility-\/related data in JSON format.
