\doxysubsection{Detailed Description}
Represents a residential building in the city. 

The \mbox{\hyperlink{classResidential}{Residential}} class is a derived class from the \mbox{\hyperlink{classBuilding}{Building}} base class. It encapsulates the properties and behaviors specific to residential buildings, including capacity management and resource collection.
