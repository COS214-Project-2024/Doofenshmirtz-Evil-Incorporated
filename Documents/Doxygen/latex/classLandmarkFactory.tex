\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classLandmark}{Landmark}} buildings in the city simulation. 

A factory class responsible for creating \mbox{\hyperlink{classLandmark}{Landmark}} buildings.

The \mbox{\hyperlink{classLandmarkFactory}{Landmark\+Factory}} class implements the factory method to create instances of the \mbox{\hyperlink{classLandmark}{Landmark}} building type and provides resource cost details for building landmark structures.

The \mbox{\hyperlink{classLandmarkFactory}{Landmark\+Factory}} class derives from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides the implementation details for creating a \mbox{\hyperlink{classLandmark}{Landmark}} building, retrieving its construction cost, and specifying the resources required.
