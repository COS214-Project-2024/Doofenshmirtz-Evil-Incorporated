Detailed Description

Factory class for creating Industrial buildings in the city simulation.

A factory class responsible for creating Industrial buildings.

The IndustrialFactory class implements the factory method to create instances of the Industrial building type and provides resource cost details for building industrial structures.

The IndustrialFactory class derives from BuildingFactory and provides the implementation details for creating an Industrial building, retrieving its construction cost, and specifying the resources required.
