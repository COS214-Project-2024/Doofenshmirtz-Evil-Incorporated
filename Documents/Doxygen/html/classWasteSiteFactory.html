Detailed Description

Factory class for creating Waste Site buildings in the city simulation.

A factory class responsible for creating Waste Site buildings.

The WasteSiteFactory class implements the factory method to create instances of the Waste Site building type and provides resource cost details for building waste management sites.

The WasteSiteFactory class derives from BuildingFactory and provides the implementation details for creating a Waste Site building, retrieving its construction cost, and specifying the resources required.
