Detailed Description

Factory class for creating Power Plant buildings in the city simulation.

A factory class responsible for creating Power Plant buildings.

The PowerPlantFactory class implements the factory method to create instances of the Power Plant building type and provides resource cost details for building power plants.

The PowerPlantFactory class derives from BuildingFactory and provides the implementation details for creating a Power Plant building, retrieving its construction cost, and specifying the resources required.
