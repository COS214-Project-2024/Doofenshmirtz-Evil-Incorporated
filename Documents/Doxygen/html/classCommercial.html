Detailed Description

Represents a commercial building in the city simulation.

Represents a commercial building in the city.

The Commercial class inherits from Building and manages resources, salary payments to residents, and notifications for front-end updates.

The Commercial class inherits from the Building class and implements functionalities specific to commercial buildings, such as updating salaries for residents, collecting resources, and providing JSON representation.
